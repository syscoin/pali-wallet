---
title: Upgrade from older Pali versions
---

If you are migrating from an older Pali version, the safest path is to import your existing seed phrase into the current wallet version.

## Migration steps

1. Open the old Pali wallet.
2. Open General settings.
3. Choose Wallet seed phrase.
4. Enter your password and copy the seed phrase.
5. Install or open the current Pali wallet.
6. Choose Import using wallet seed phrase.
7. Paste the seed phrase.
8. Set a new wallet password.
9. Verify your expected accounts and networks.

## Passkey accounts after migration

Passkey smart accounts are newer than the old docs. If you used passkey accounts and local metadata is missing, use the passkey recovery flow in Settings. Recovery may require the same authenticator credential and, for some required sponsor policies, the sponsor service URL.
