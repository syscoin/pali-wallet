---
title: 古いPaliバージョンからのアップグレード
---

古いPaliバージョンから移行する場合、最も安全な方法は、既存のseed phraseを現在のウォレットバージョンにインポートすることです。

## 移行手順

1. 古いPaliウォレットを開きます。
2. General settingsを開きます。
3. Wallet seed phraseを選択します。
4. パスワードを入力し、seed phraseをコピーします。
5. 現在のPaliウォレットをインストールまたは開きます。
6. Import using wallet seed phraseを選択します。
7. seed phraseを貼り付けます。
8. 新しいウォレットパスワードを設定します。
9. 期待するアカウントとネットワークを確認します。

## 移行後のpasskeyアカウント

Passkeyスマートアカウントは古いドキュメントより新しい機能です。passkeyアカウントを使用していてローカルメタデータがない場合は、Settingsのpasskey復元フローを使用してください。復元には同じ認証器credentialが必要になる場合があります。
