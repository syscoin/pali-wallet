---
title: Обновление со старых версий Pali
---

Если вы мигрируете со старой версии Pali, самый безопасный путь — импортировать существующую seed phrase в текущую версию кошелька.

## Шаги миграции

1. Откройте старый Pali wallet.
2. Откройте General settings.
3. Выберите Wallet seed phrase.
4. Введите пароль и скопируйте seed phrase.
5. Установите или откройте текущий Pali wallet.
6. Выберите Import using wallet seed phrase.
7. Вставьте seed phrase.
8. Задайте новый пароль кошелька.
9. Проверьте ожидаемые аккаунты и сети.

## Passkey accounts после миграции

Passkey smart accounts новее старой документации. Если вы использовали passkey accounts, а локальные metadata отсутствуют, используйте recovery flow passkey в Settings. Для восстановления может потребоваться тот же authenticator credential и, для некоторых required sponsor policies, URL sponsor service.
