title: Начало работы для пользователя
---

Pali позволяет управлять EVM аккаунтами, Syscoin UTXO аккаунтами и passkey smart accounts из одного расширения.

## Базовая настройка

1. Установите расширение Pali.
2. Создайте новый кошелек или импортируйте существующую seed phrase.
3. Задайте надежный пароль.
4. Сделайте offline-резервную копию вашей seed phrase.
5. Выберите сеть, которую хотите использовать.
6. Подключайтесь только к dapps, которым доверяете.

## Подключение к dapp

Когда сайт запрашивает доступ, Pali открывает popup подключения, который показывает сайт и позволяет выбрать аккаунт. Dapp получает только адрес подключенного аккаунта и одобренное состояние провайдера.

Pali сохраняет подключения по сайтам. Вы можете подключать разные сайты к разным аккаунтам, но у каждого сайта одновременно есть один активный аккаунт.

## EVM аккаунты

Используйте EVM аккаунты для Ethereum-совместимых цепей, Rollux, Syscoin NEVM и dapps, ожидающих поведение кошелька в стиле MetaMask.

EVM dapps могут запрашивать:

- доступ к аккаунту
- транзакции
- personal signatures
- typed data signatures
- запросы на отслеживание токенов
- запросы на добавление/переключение цепи
- batch call requests

## UTXO аккаунты

Используйте UTXO аккаунты для Syscoin UTXO и Bitcoin-style transaction flows. UTXO dapps могут запрашивать состояние с учетом xpub, change addresses, подписание PSBT и broadcast транзакций.

## Passkey smart accounts

Passkey accounts — это smart accounts, контролируемые WebAuthn credentials. Они могут быть полезны для onboarding, управляемого институцией, восстановления аккаунта и sponsored execution. Некоторые passkey accounts являются counterfactual до своей первой deployment transaction.
