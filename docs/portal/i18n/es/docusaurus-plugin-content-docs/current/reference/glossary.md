title: Glosario
---

## Familia de cuenta

El modelo de cuenta de billetera usado por una red. Las cuentas EVM están basadas en cuentas. Las cuentas Syscoin UTXO y de estilo Bitcoin están basadas en UTXO.

## EIP-1193

La interfaz estándar de proveedor JavaScript usada por billeteras Ethereum mediante `provider.request()`.

## EIP-6963

Un estándar de descubrimiento multi-wallet que permite a las dapps descubrir proveedores sin depender solo de `window.ethereum`.

## Cuenta inteligente passkey

Una cuenta inteligente EVM controlada por una credencial WebAuthn en vez de una clave privada EOA normal.

## PSBT

Partially Signed Bitcoin Transaction. Un formato común para coordinar la firma de transacciones UTXO.

## Servicio sponsor

Un servicio operado por una institución que puede proporcionar sponsorship de gas, soporte de relay o coautorización requerida para ejecución de cuentas inteligentes passkey.

## SPT

Syscoin Platform Token, un activo en el lado Syscoin UTXO.

## UTXO

Unspent Transaction Output. El modelo de cuenta usado por cadenas tipo Bitcoin y Syscoin UTXO.

## WebAuthn

El estándar de navegador detrás de las passkeys. Permite a los usuarios aprobar acciones criptográficas mediante autenticadores de plataforma, llaves hardware o proveedores de passkey sincronizadas.
