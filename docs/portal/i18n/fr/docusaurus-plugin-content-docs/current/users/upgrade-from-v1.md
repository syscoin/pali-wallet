---
title: Mettre à niveau depuis d'anciennes versions de Pali
---

Si vous migrez depuis une ancienne version de Pali, le chemin le plus sûr consiste à importer votre phrase de récupération existante dans la version actuelle du portefeuille.

## Étapes de migration

1. Ouvrez l'ancien portefeuille Pali.
2. Ouvrez les paramètres General.
3. Choisissez Wallet seed phrase.
4. Entrez votre mot de passe et copiez la phrase de récupération.
5. Installez ou ouvrez le portefeuille Pali actuel.
6. Choisissez Import using wallet seed phrase.
7. Collez la phrase de récupération.
8. Définissez un nouveau mot de passe de portefeuille.
9. Vérifiez vos comptes et réseaux attendus.

## Comptes passkey après migration

Les comptes intelligents passkey sont plus récents que les anciens docs. Si vous avez utilisé des comptes passkey et que les métadonnées locales manquent, utilisez le flux de récupération passkey dans Settings. La récupération peut exiger le même credential d'authenticator et, pour certaines politiques de sponsor requises, l'URL du service de sponsor.
