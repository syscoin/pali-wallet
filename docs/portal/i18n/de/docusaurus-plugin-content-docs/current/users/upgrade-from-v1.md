---
title: Upgrade von älteren Pali-Versionen
---

Wenn Sie von einer älteren Pali-Version migrieren, ist der sicherste Weg, Ihre vorhandene Seed-Phrase in die aktuelle Wallet-Version zu importieren.

## Migrationsschritte

1. Öffnen Sie die alte Pali Wallet.
2. Öffnen Sie die allgemeinen Einstellungen.
3. Wählen Sie Wallet-Seed-Phrase.
4. Geben Sie Ihr Passwort ein und kopieren Sie die Seed-Phrase.
5. Installieren oder öffnen Sie die aktuelle Pali Wallet.
6. Wählen Sie Import using wallet seed phrase.
7. Fügen Sie die Seed-Phrase ein.
8. Legen Sie ein neues Wallet-Passwort fest.
9. Verifizieren Sie Ihre erwarteten Accounts und Netzwerke.

## Passkey-Accounts nach der Migration

Passkey Smart Accounts sind neuer als die alten Dokumente. Wenn Sie Passkey-Accounts verwendet haben und lokale Metadaten fehlen, verwenden Sie den Passkey-Wiederherstellungs-Flow in den Einstellungen. Die Wiederherstellung kann dasselbe Authenticator-Credential und, für manche erforderlichen Sponsor-Policies, die Sponsor-Service-URL erfordern.
